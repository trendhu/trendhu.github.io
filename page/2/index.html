docker logs 说明

发表于 2020-01-05 | 更新于: 2020-01-05 | 分类于 运维 |

描述

获取容器的日志

用法

选项

名字 简写 默认 描述
--details 显示提供给日志的额外细节
--follow -f 按照日志输出 (查看实时日志)
--since 自时间戳（例如2013-01-02T13：23：37）
或相对（例如42分钟42分钟）开始显示日志
（日志显示起始时间）
--tail --all 从日志末尾显示的行数
--timestamps -t 显示时间戳
--until 在时间戳（例如2013-01-02T13：23：37）
或相对（例如42分钟42分钟）之前显示日志
（日志显示结束时间）
